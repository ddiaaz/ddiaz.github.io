SIR COVID-19 Model: Modeling the Coronavirus Pandemic

This project uses the Fourth Order Runge-Kutta Method to solve for an SIR model that represent the coronavirus pandemic. Each differential equation in the SIR model corresponds to the Susceptible, Infected, and Recovered populations of a region and extrapolates their values over time.

Through this model, I developed a few scenarios on what the pandemic would look like under an infectious variant of the virus, if no one in LA County were to be vaccinated, and if 70% of LA County was fully-vaccinated. Each scenario is visualized with a detailed graph.

Weather Data Structure

In this project, I developed data structures to build a Weather Database that manages temperature, wind, humidity, and location data with C++. This includes the Binary Search Tree, Hash Table, and Singly Linked-List data structures.

Through the program menu, the user can import data manually or through a text file, search for data, delete data, find data by a hash key, and view efficiency measures.
